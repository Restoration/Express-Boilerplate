# Express BoilerPlate

## Install

### Use Docker
```
$ cd app
$ sh setup.sh
```

### Use your directory
```
$ cd app
$ npm i
$ node app.js
```

## Functions
- Login
- CRUD
- JWT Auth

## Requirement
- express.js
- passport
- mongoDB
- Json Web Token
